ShelfPack #allocate #pack #resize

ShelfPack(options)

index.js

Uses the Shelf Best Height Fit algorithm from http://clb.demon.fi/files/RectangleBinPack.pdf

Parameters

  • Object options :
    • number = options.width (default 64)

      Initial width of the sprite

    • number = options.height (default 64)

      Initial width of the sprite

    • boolean = options.autoGrow (default false)

      If true, the sprite will automatically grow

Examples

var sprite = new ShelfPack({width: 64, height: 64});
